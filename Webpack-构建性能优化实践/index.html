# 性能优化方向总览

  • 缓存： cache.type = 'filesystem' 、 babel-loader 的 cacheDirectory 。
  • 并行： thread-loader 或 esbuild-loader / swc-loader 。
  • 代码分割： optimization.splitChunks 与 dynamic import 。
  • 体积压缩： TerserPlugin 、 CssMinimizerPlugin 、图片压缩。

# 图片与静态资源优化

  • 使用内置 asset 模式，按阈值自动转为 base64 或文件。
  • 在生产环境开启 image-minimizer-webpack-plugin ；也可在 CI 中预处理图片。

# 构建速度排查

  • 开启 stats 或使用 webpack-bundle-analyzer 分析包体与模块依赖。
  • 关注长耗时 loader/plugin，按需替换为更快实现（如 esbuild/swc）。
