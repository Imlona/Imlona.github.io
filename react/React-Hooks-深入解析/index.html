# 常用 Hooks

  • useState：管理本地状态。
  • useEffect：副作用处理（订阅/回收、数据请求）。
  • useMemo：记忆化计算，避免昂贵计算重复执行。
  • useCallback：记忆化函数，降低子组件不必要渲染。
  • useRef：持久化引用，用于 DOM 或保存可变值。

# useEffect 示例

useEffect(() => {
  const id = setInterval(() => {
    // do something
  }, 1000);
  return () => clearInterval(id); // 清理
}, []);

# 自定义 Hook

function useFetch(url) {
  const [data, setData] = useState(null);
  useEffect(() => {
    fetch(url).then(r => r.json()).then(setData);
  }, [url]);
  return data;
}

# 注意事项

  • 仅在函数最外层调用 Hooks；不要在循环、条件、嵌套函数中调用。
  • 所有依赖必须在依赖数组中声明，保证一致性与可维护性。
