# 渲染与重渲染控制

  • 使用 React.memo 缓存纯组件。
  • 结合 useMemo 与 useCallback ，稳定 props 与函数引用。
  • 合理设置 key ，避免不必要的 diff。

# 列表优化

  • 虚拟化长列表（如 react-window 、 react-virtualized ）。
  • 避免在 render 中创建大对象或匿名函数。

# 网络与图片优化

  • 采用缓存策略与合理的请求合并。
  • 使用现代图片格式（WebP/AVIF）与懒加载。

# 监控与分析

  • 使用 React Profiler 分析渲染耗时。
  • 在生产环境接入前端性能监控（FCP、LCP、TTI）。
